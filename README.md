# 💰 Gamified Personal Finance Tracker

A dynamic, web-based personal finance application that turns monthly budget tracking into a gamified experience. Users can set budgets, track expenses, and earn "Discipline Points" to level up by staying within their financial limits.

## ✨ Features

- **Interactive User Profile:** Welcomes the user dynamically based on their input.
- **Budget Management:** Allows users to set a monthly limit and add expenses in real-time.
- **Dynamic Visual Feedback:** Features a progress bar that updates instantly and turns red when the budget exceeds the 80% critical threshold.
- **Gamification System:** - Earn Discipline Points for closing the month under budget.
  - Lose points for exceeding the budget.
  - Level up upon reaching 150 points.
- **Month Reset:** A single click calculates the final score, updates the user's level, and resets the board for a new month.

## 🛠️ Technologies Used

- **HTML5** for page structure
- **Tailwind CSS** for modern, responsive styling
- **Vanilla JavaScript (DOM Manipulation)** for the core logic and dynamic updates
